## Setup

Install dependencies

```bash
yarn install
```

Create a .env file
```bash
VITE_API_URL=https://6141b2f5357db50017b3dc1a.mockapi.io/api/v1
```

## Development

Start the development server on http://localhost:3000

```bash
yarn dev
```
